Data model:
 - Opinion Metadata table:
     - Primary key (UUID) (int?)
     - Case number (text)
     - Case name (text)
     - Date (date)
     - Court (text)
     - URL of the case on the court site (text)
     - text of the case (text)
     - local copy of the file (PDF) (blob)
 - Except table:
     - UUID (int?)
     - smart except (text)
     - user-generated except (text)
 - Judge table:
     - UUID (int?)
     - Majority opinion writer (text)
     - Panelist (text)
 - Party table:
     - UUID (int?)
     - Party name (text)
 - Users table:
     - UUID (int?)
     - First name (text)
     - Last name (text)
     - Username (text)
     - Password (text)
     - Email address (text)
 - Bar table:
     - UUID (int?)
     - Bar membership (text)
 - Alerts table:
     - UUID (int?)
     - Alert (text)
     - Private? (boolean)
     - Rate (text)
     - Plaintext email? (boolean)
     - Send alert on no hits (boolean)
