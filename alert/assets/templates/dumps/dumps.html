XML Data Dumps

For hackers and legal analysts, we provide XML dumps of our data. We generate these files when they are requested, so some files may be very fast, while others may need to be created for you, which can take a moment. Dump files are available by the year, month and day. For each time period, individual dump files are available for each court, as well as a single file containing opinions from all courts.

The XML information in the dumps should be self-explanatory, and can be discussed in our forum.

All time stamps are Pacific Standard Time.

Using the Dumps

Annual, monthly or daily dumps can be accessed at http://courtlistener.com/dump-api/year/month/day/court.xml.gz.

For example, let's look at ways to access the First Circuit of Appeals (ca1):

  • All ca1 cases from 2009: http://courtlistner.com/dump-api/2009/ca1.xml.gz
  • All ca1 cases from June 2009: http://courtlistner.com/dump-api/2009/06/ca1.xml.gz
  • All ca1 cases from June 9, 2009: http://courtlistner.com/dump-api/2009/06/09/ca1.xml.gz

If you would like all documents for a given time period, you can use all for the court name. For example, http://courtlistener.com/2009/06/09/all.xml.gz returns all of the cases from June 9, 2009 (across all courts).
